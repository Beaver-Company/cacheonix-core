Overview
--------

Directory "annotations" is a root for Cacheonix module responsible for Cacheonix annotations.

Java Version
------------

Cacheonix annotations use Java 5.

Module Structure
----------------

Cacheonix annotations include the following directories:

core - Source Java files for Cacheonix annotations.
test - Test sources and data for Cacheonix annotations.


Dependencies
------------

Cacheonix annotations depend on Cacheonix core. Other modules do not depend on Cacheonix annotations.
