Contains Cacheonix cache controller and cache factory for iBatis 2.3.0

Configuring Cacheonix Cache Model For iBatis

Configuration overview:

  1. Create cacheonix-config.xml
  2. Add a cache configuration to cacheonix-config.xml
  3. Put cacheonix-config.xml into the class path
  4. Provide a name of a cache from cacheonix-config.xml in the iBatis mapping

Visit Cacheonix documentation, section Configuring Distributed iBatis Cache for detailed information on configuring Cacheonix iBatis cache controller.

@since Cacheonix 1.0
